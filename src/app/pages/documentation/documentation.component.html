Hide/Show
For hide and show in the footer of table there is 3 lined icon.while clicking on it. You can select columns to show. Moreover, there is select All button for selecting all columns. Addition to that.when user select a columns and leave the page next time when user comes that page.it will show selected columns.(page remembers user configurations.)
Search/Filter and sorting
For search/filter there is a filter icon in every column next to title. You can open and filter data based on that column. For sorting you have to click on column itself. Default latest content is ascending that's why there is down icon there.You can change by clicking on coloumn.
Delete and Export button
User can select multiple rows to delete(it only deletes in the client side) and export to excel(xlss) file. While there is not selected column export button will export all the data.
Loading data
For loading data from file.I used service and help of rxjs library. "RxJS" is a library for reactive programming using Observables, to make it easier to compose asynchronous or callback-based code.
Language
Website translated into 2 languages: English and Turkish.You can use translation icon in the top right corner of website(in the header) It also saves the user language for next time.

Build
For building website, "ng build --prod" --aot=true|false (Build using Ahead of Time compilation.) Copy everything within the output folder (dist/ by default) to a folder on the server. Configure the server to redirect requests for missing files to index.html More info

The Angular CLI downloads and install everything you need to test an Angular application with the Jasmine test framework. The project you create with the CLI is immediately ready to test.
